ce readme concerce les aplication fr-dos
A propos:
-les app fr-dos sont en batch (la comande dev [nom de l'apliction] permet de developer directement via le format .exe en batch avec notepad)
-les app son enregistrer au format .exe mais ne sont pas executable leur contenu est juste envoyer dans temp.bat (le tout est fait automatiquement via la comande prg [nom de l'aplication])
-les app sont modifiable par n'importe qui via la comande dev [nom de l'apliction]
Résumer du fonctionement:
dev>fr-dos>notepad (nom de l'app entré).exe>sauvegarder dans c:\fr-dos\(ton nom d'utilisateur)\
prg>fr-dos>(nom de l'aplication).exe > temp.bat >fr-dos >lancement de temp.bat>fin de temp.bat>supresion de temp.bat
